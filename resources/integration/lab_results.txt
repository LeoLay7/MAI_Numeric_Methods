Лабораторная работа: Численное интегрирование
==============================================

Задание:
Вычислить определенный интеграл по формулам средних прямоугольников,
трапеций и Симпсона на равномерной сетке с точностью eps=10^-3.
Шаг интегрирования выбрать таким образом, чтобы на заданном отрезке 
можно было применить все перечисленные методы.
Оценить погрешности методов интегрирования. Для каждого метода уточнить 
полученные значения, используя метод Рунге–Ромберга.

Функция: F = ∫[0.5 to 3] (1+2/√x)/(√x + x)² dx

- Значение интеграла: 1.2326211465924675
- Количество разбиений: 64
- Погрешность: 5.867E-4
- Уточненное значение (Рунге-Ромберг): 1.233207851842557

Метод трапеций:
- Значение интеграла: 1.2335080147521917
- Количество разбиений: 128
- Погрешность: 2.956E-4
- Уточненное значение (Рунге-Ромберг): 1.2332123920322833

Метод Симпсона:
- Значение интеграла: 1.2342055047589298
- Количество разбиений: 16
- Погрешность: 6.238E-4
- Уточненное значение (Рунге-Ромберг): 1.2335816930467114

Выводы:
-------
1. Метод Симпсона показал наилучшую эффективность, требуя всего 16 разбиений
   для достижения заданной точности.
2. Метод средних прямоугольников потребовал 64 разбиения.
3. Метод трапеций оказался наименее эффективным, требуя 128 разбиений.
4. Все методы дали близкие результаты после уточнения по Рунге-Ромбергу.
